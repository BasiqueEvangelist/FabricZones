swapInventories

fun swapInventories(inBuilderMode: Boolean): Boolean
Content copied to clipboard

Set's the player's active inventory to their saved inventory, and saves the previously active inventory. The inventory is saved with the "BuilderInventory" tag on the player's data. For an extra measure of "security," there is a boolean "builderInven" which marks the saved inventory as either the inventory saved from builder mode (true) or the inventory saved from normal play (false).

Return

Whether or not the inventory was successfully swapped

Parameters

inBuilderMode

if this boolean matches the "builderInven" boolean of the saved inventory, then it will successfully swap inventories, otherwise it will cancel
